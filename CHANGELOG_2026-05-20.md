# CHANGELOG: 20 Мая 2026

Добавлено новое поле в ответ метода `GetPublic` клиентского справочника (`DictionaryService`) для нужд команды по верстке.

## 🚀 Новые поля и изменения в API

### dictionary.proto

В структуру сообщения `PublicResponse` добавлено новое булево поле, сигнализирующее о переходе на новую версию маркетинга (V2):

```protobuf
message PublicResponse {
    // ... существующие поля ...

    // Флаг перехода на новую версию маркетинга (V2: выплата реферального бонуса только на 1-м уровне).
    bool marketing_v2 = 16;
}
```

### 💻 Пример для Frontend

Команда по верстке теперь может использовать это поле для динамического скрытия/показа виджетов или изменения их поведения в зависимости от того, активна ли новая версия маркетинга (V2).

**Пример использования ответа:**
```javascript
const publicData = await DictionaryServiceClient.GetPublic({});
if (publicData.marketingV2) {
    // Включить логику/виджеты для новой версии маркетинга V2
    showV2Widgets();
} else {
    // Использовать стандартное поведение V1
    showLegacyWidgets();
}
```
