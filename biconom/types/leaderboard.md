# Leaderboard — Базовые типы лидерских досок

## Описание

`Leaderboard` предоставляет универсальные типы данных для подсистемы ранжирования участников
(Leaderboard Engine). Эта подсистема обеспечивает хранение, ранжирование (сложность $O(\log N)$)
и извлечение данных участников на основе их баллов.

Типы из этого файла переиспользуются в модулях, работающих с рейтингами (например, `Arena`),
чтобы избежать дублирования структуры `Entry` или `Board`.

## Типы

| Тип | Назначение |
|-----|------------|
| `Leaderboard.Id` | Идентификатор доски |
| `Leaderboard.Status.Id` | Жизненный цикл доски (ACTIVE, FROZEN, ARCHIVED) |
| `Leaderboard.Board` | Метаданные доски: ID, статус, валюта баллов, кол-во участников |
| `Leaderboard.Entry` | Запись участника: user_id (или distributor_id), баллы, текущий ранг и метки времени |

## Особенности `Entry`
- `value`: строка (mantissa), представляющая количество заработанных баллов. Строковое представление
  гарантирует отсутствие потерь точности при передаче (например, `u128`).
- `rank`: натуральное число от 1 и выше. Каждому участнику присваивается уникальный ранг (один ранг — один пользователь).
