# Сервис: MarketingService (Admin)

## 1. Описание

**`biconom.admin.marketing.MarketingService`** предоставляет административный интерфейс для управления маркетинговыми активностями, лицензиями и слотами. В отличие от клиентского API, методы здесь доступны только сотрудникам компании с соответствующими правами.

## 2. Методы API

### `GrantSubscription`
- **Назначение**: Позволяет администратору "подарить" (выдать бесплатно) подписку или лицензию конкретному слоту или дистрибьютору. Технически это эквивалентно покупке пользователем пакета, но списания средств не происходит (оплачивается "Компанией").
- **Входные параметры**:
    - `plan_id`: Идентификатор тарифного плана (`license_id` + `license_entity_id`).
    - `target`: Цель назначения — либо конкретный `slot_id`, либо `distributor_id`.
    - `admin_comment`: Примечание для логов аудита.
- **Возвращаемое значение**: Обновленный объект `MarketingSlot` с актуальным состоянием лицензии.

## 3. Права доступа и безопасность
- Требуется роль администратора (например, `MARKETING_ADMIN` или `SUPER_ADMIN`).
- Все действия логируются в Audit Log с указанием `admin_comment`.

## 4. Сценарии использования
- **Подарок от компании**: Начисление приветственного пакета топ-лидерам при старте.
- **Компенсация**: Выдача лицензии в качестве извинения за технические сбои.
- **Тестирование**: Выдача лицензий тестовым аккаунтам для проверки механик маркетинга.
